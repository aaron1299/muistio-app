Jaa GitHubin kautta

Yhdistä yksityiseen GitHub-repoon, niin sinä ja kaverisi näette ja muokkaatte samoja muistiinpanoja. Ei palvelinta, ei ylläpitoa.

Näin liityt (kerran):
  1. Luo henkilökohtainen token: github.com/settings/personal-access-tokens/new
  2. Valitse Resource owner = repon omistaja, Repository access = Only select repositories → tämä repo.
  3. Permissions → Repository → Contents → Read and write. Luo token ja kopioi se.
  4. Liitä token alle. Token tallentuu vain tämän selaimen muistiin, ei mihinkään muualle.
Repon tulee olla yksityinen, ja molemmat lisätään sen jäseniksi.
